# Export Specifications

## PDF Export

**Content:** Full report with all sections  
**Format:** Branded template with headers, footers  
**Evidence:** Inline citations converted to footnotes  
**Images:** Included at optimized resolution

## PowerPoint Export

**Slides Generated:**
1. Title slide
2. Executive summary (key findings)
3. Brand platform summary
4. Archetype & narrative
5. Voice & tone
6. Visual identity summary
7. Competitive positioning map
8. Key tensions
9. Recommendations (3 slides)
10. Appendix: Evidence index

## Data Export

**Format:** JSON or CSV  
**Content Options:**
- Full machine layer outputs
- Summary data only
- Specific entity/section
